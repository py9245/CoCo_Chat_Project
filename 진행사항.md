- `rest_framework.authtoken`을 활성화하고 `REST_FRAMEWORK` 기본 인증/권한 설정을 추가했습니다.
- 사용자 회원가입·로그인·프로필·로그아웃 엔드포인트(`/api/auth/*`)를 구현했고, 중복 검사 및 토큰 발급 로직을 직렬화기로 분리했습니다.

- 메인 SPA에 회원가입/로그인 UI, 로그인 상태 표시, 에러/성공 메시지 등을 추가했습니다.
- 스타일 파일을 업데이트해 인증 폼/버튼 UI를 정리하고 전체 빌드(`npm run build`)를 다시 생성했습니다.
